# Новый Workflow - Контент-План с Одобрением

## 🎯 Проблема которую решаем

**Старый workflow:**
1. Генератор создает посты
2. Автоматически добавляет их в календарь
3. Пользователь не видит что было создано до навигации

**Проблемы:**
- ❌ Нет возможности просмотреть план перед сохранением
- ❌ Нет контроля над тем, что добавляется в календарь
- ❌ Race condition при асинхронном сохранении state
- ❌ Пользователь не может отменить генерацию

---

## ✅ Новый Workflow

### Шаг 1: Генерация контент-плана
**Страница:** `/generate` (Генератор месяца)

Пользователь выбирает:
- Месяц
- Дни недели
- Время публикаций
- Платформы (TikTok/Instagram)
- Цели контента

Нажимает **"Сгенерировать контент-план"**

---

### Шаг 2: Просмотр и одобрение ⭐ НОВОЕ!
**Страница:** `/review` (Просмотр контент-плана)

После генерации пользователь автоматически переходит на страницу просмотра где видит:

#### Сводка плана:
- Всего постов
- Используемые платформы
- Количество форматов
- Период публикаций

#### Таблица всех постов:
| # | Дата | Время | Платформа | Формат | Цель | Тема | Хэштеги | Промпт |
|---|------|-------|-----------|--------|------|------|---------|--------|
| 1 | 02 июн | 09:00 | TikTok | Video | Охват | ... | #тег1 #тег2 | ✓ Готов |

#### Доступные действия:

**🟢 Одобрить и добавить в календарь**
- Сохраняет все посты в приложение
- Добавляет месячный план
- Переходит в календарь
- Посты сразу видны в календаре

**🔴 Отклонить**
- Отменяет генерацию
- Возвращает к генератору
- Посты НЕ сохраняются

**⬅️ Вернуться к генератору**
- Возвращает к настройкам
- Можно изменить параметры и сгенерировать заново

---

### Шаг 3: Работа с календарем
**Страница:** `/calendar`

После одобрения плана:
- Все посты отображаются в календаре
- Мини-карточки показывают время, платформу, цель
- Клик на карточку → модальное окно
- Редактирование через модальное окно

## 🔧 Технические детали

### Передача данных через React Router

```typescript
// MonthGenerator.tsx
navigate('/review', {
  state: {
    posts: postsWithSEO,        // Массив сгенерированных постов
    monthlyPlan: monthlyPlan,   // Данные месячного плана
  },
});
```

```typescript
// ContentPlanReview.tsx
const location = useLocation();
const state = location.state as { posts?: Post[]; monthlyPlan?: any };
```

### Безопасное сохранение

Посты сохраняются только после явного одобрения:

```typescript
const handleApprove = () => {
  addPosts(generatedPosts);      // Массовое добавление
  addMonthlyPlan(monthlyPlanData);
  
  setTimeout(() => {
    navigate('/calendar');        // Навигация после сохранения
  }, 500);
};
```

Основная страница просмотра и одобрения контент-плана.
