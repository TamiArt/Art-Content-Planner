# Art Content Planner

Личный контент-оператор для художника, который за 10 минут генерирует контент-план на месяц.

## Основные возможности

### Генератор контент-плана
- Автоматическое создание постов на месяц с учетом воронки и целей
- Выбор дней недели, времени публикаций, платформ (TikTok/Instagram)
- Чередование целей: охват → доверие → заявка/продажа
- Защита от повторов тем, форматов, хуков и CTA
- Проверка существующих постов с опцией добавить или заменить план
- Автоматическая генерация промптов для каждого поста
- Уведомление об успешном создании плана

### Календарь публикаций — 4 режима отображения
1. **Месяц** — визуализация всех публикаций с мини-карточками в каждой дате
   - Время публикации
   - Платформа (TT/IG)
   - Тема поста (укороченная)
   - Цель контента (цветной бейдж)
   - Клик на карточку открывает модальное окно

2. **Неделя** — подробный вид ближайших 7 дней
   - Время, платформа, формат
   - Полная тема поста
   - Цель контента
   - Статус выполнения

3. **Список** — табличное представление всех постов
   - Дата, время, платформа, цель, тема, статус
   - Сортировка по дате
   - Быстрые действия

4. **Канбан** — посты по статусам
   - Идея → Промпт готов → Текст готов → Визуал готов → Запланировано → Опубликовано

### Модальное окно поста
При клике на карточку в календаре открывается полная информация:
- Дата, время, платформа, формат, цель, статус
- Идея поста, CTA, хэштеги
- Кнопки действий:
  - **Скопировать промпт** — промпт автоматически генерируется если его нет
  - **Редактировать** — переход на полный редактор
  - **Перегенерировать** — новый промпт и SEO-ключи
  - **Изменить дату/время** — перенос поста на другую дату
  - **Удалить** — удаление поста

### Цветовая индикация целей
- **Охват** — синий (#3b82f6)
- **Вовлечение** — фиолетовый (#8b5cf6)
- **Доверие** — зелёный (#10b981)
- **Заявка** — оранжевый (#f59e0b)
- **Продажа** — красный (#ef4444)

### Промпт-билдер
- Генерация промптов для ChatGPT/Claude/Gemini
- Учёт стиля, тона и фокуса контента
- Включает: хуки, визуальный сценарий, структуру текста, CTA, LSI-ключи, хэштеги

### Дополнительно
- **Банк идей** — быстрая запись и организация идей для контента
- **Аналитика** — ручной ввод метрик и автоматический расчет показателей
- **Мои картины** — каталог работ для упоминания в контенте
- **Услуги и офферы** — готовые предложения для продающих постов
- **Экспорт/импорт JSON** — резервное копирование всех данных

## Технологии

- React 18 + TypeScript
- React Router 7 для навигации

- LocalStorage для хранения данных

## Основные модули

### 1. Генератор контента (`src/utils/contentGenerator.ts`)
Создает контент-план по правилам:
- День → Цель → Этап воронки → Тема → Формат → Промпт
- Защита от повторов тем, форматов, хуков, CTA
- Чередование целей: охват → доверие → продажа

### 2. Промпт-билдер (`src/utils/promptBuilder.ts`)
Генерирует детальные промпты для ИИ с учетом:
- Платформы (TikTok/Instagram)
- Формата контента
- Стиля и тона
- Цели и этапа воронки

### 3. SEO/LSI (`src/utils/seoKeywords.ts`)
Подбирает ключевые слова и хэштеги из предустановленных кластеров:
- Интерьерные скетчи
- Картины маслом
- Картина в интерьере
- Процесс рисования
- Ошибки и обучение
- Продажа и заявки

### 4. Аналитика (`src/utils/analytics.ts`)
Рассчитывает метрики:
- Engagement Rate, Save Rate, Share Rate
- Лучшие темы, форматы, дни, время
- Конверсия в подписки и заявки

## Быстрый старт

1. **Генератор месяца** → выберите месяц, дни, время, платформы и цели → "Сгенерировать контент-план"
2. **Просмотр плана** → увидите таблицу со ВСЕМИ сгенерированными постами
3. **Одобрение** → нажмите "Одобрить и добавить в календарь" или "Отклонить"
4. После одобрения откроется **Календарь** с добавленными постами
5. В календаре кликайте на мини-карточки постов для просмотра деталей
6. В модальном окне нажмите **"Скопировать промпт"**
7. Вставьте промпт в ChatGPT/Claude/Gemini и получите готовый контент
8. После публикации добавьте аналитику в редакторе поста
9. Отслеживайте лучшие темы, форматы и время в разделе **Аналитика**

## Рабочий процесс

### Шаг 1: Генерация плана на месяц
- Откройте "Генератор месяца"
- Выберите месяц (например, июль 2026)
- Выберите дни недели (понедельник, среда, пятница)
- Добавьте время публикаций: каждое время создаёт отдельный пост в выбранный день (например, пост и сторис в разное время)
- Отметьте платформы (TikTok и/или Instagram)
- Выберите цели (охват, доверие, заявка)
- Нажмите "Сгенерировать контент-план"

Если на выбранный месяц уже есть посты:
- Введите `1` чтобы добавить новые посты к существующим
- Введите `2` чтобы заменить весь план
- Введите `3` для отмены

### Шаг 2: Просмотр календаря
После генерации вы автоматически попадёте в календарь. Выберите режим:

**Месяц** — для общего обзора всех публикаций
- Видны все даты месяца
- Мини-карточки показывают время, платформу, тему и цель
- Несколько постов в один день показываются отдельными карточками

**Неделя** — для детального планирования на 7 дней
- Подробные карточки с форматом и статусом
- Удобно для текущей недели

**Список** — для табличного просмотра
- Вся информация в одной таблице
- Быстрая сортировка и фильтрация

**Канбан** — для отслеживания прогресса
- Посты группируются по статусам
- Видно что на какой стадии

### Шаг 3: Работа с постом
Кликните на любую карточку в календаре:
- Откроется модальное окно с полной информацией
- Нажмите "Скопировать промпт" (промпт генерируется автоматически если его нет)
- Вставьте промпт в AI-чат (ChatGPT, Claude, Gemini)
- Скопируйте ответ ИИ обратно в пост через "Редактировать"
- Измените статус на "Текст готов"

### Шаг 4: Изменение даты/времени
Если нужно перенести пост:
- Откройте модальное окно поста
- Нажмите "Изменить дату/время"
- Выберите новую дату и время
- Пост автоматически переместится в календаре

### Шаг 5: Аналитика после публикации
После публикации поста:
- Откройте пост через календарь или редактор
- Добавьте метрики: просмотры, лайки, комментарии, сохранения, заявки
- Приложение автоматически рассчитает: Engagement Rate, Save Rate, Conversion
- Перейдите в "Аналитика" чтобы увидеть лучшие темы, форматы и время

## Исправленные баги

### ✅ Посты не сохранялись в календарь (исправлено 06.06.2026)
**Проблема:** После генерации и одобрения контент-плана посты не появлялись в календаре.

**Причина:** Функции обновления состояния в `AppContext` использовали устаревшее значение `data` из замыкания вместо актуального состояния React.

**Решение:** Все функции обновления переписаны с использованием функционального обновления `setData(prevData => ...)`, что гарантирует работу с актуальным состоянием.

**Детали:** См. `BUGFIX_STATE_UPDATE.md`

---

### ✅ Посты не отображаются в календаре из-за часового пояса (исправлено 06.06.2026)
**Проблема:** Посты были в списке, но не отображались в календаре. Особенно затрагивало пользователей в UTC+3 и выше (Россия, Азия).

**Причина:** Использование `toISOString().split('T')[0]` конвертировало локальную дату в UTC, из-за чего даты смещались на день назад.

**Решение:** Создана функция `formatDateLocal()` для форматирования дат без конвертации в UTC. Исправлено в `contentGenerator.ts` и `CalendarView.tsx`.

**Детали:** См. `TIMEZONE_BUG_FIX.md`
