# Руководство по использованию Art Content Planner

## Основной рабочий процесс

### 1. Первый запуск

При первом запуске приложение создаёт пустое хранилище данных. Рекомендуем начать с генерации контент-плана.

### 2. Генерация контент-плана

**Путь:** Dashboard → "Генератор месяца" или боковое меню → "Генератор месяца"

**Параметры:**
- **Месяц:** выберите месяц для планирования (например, 2026-07 для июля 2026)
- **Частота:** сколько постов в неделю (по умолчанию 3)
- **Дни недели:** отметьте дни публикаций (например, Пн, Ср, Пт)
- **Время:** добавьте время для каждого дня (например, 09:00, 15:00, 18:00)
- **Платформы:** TikTok и/или Instagram
- **Цели:** выберите цели которые будут чередоваться (охват, вовлечение, доверие, заявка, продажа)

**Процесс:**
1. Заполните все поля
2. Нажмите "Сгенерировать контент-план"
3. Если на этот месяц уже есть посты, выберите:
   - `1` — добавить новые посты
   - `2` — заменить весь план
   - `3` — отмена
4. Приложение создаст посты и покажет сообщение: "Контент-план создан: X публикаций добавлено в календарь"
5. Вы автоматически перейдёте в календарь

### 3. Работа с календарём

**4 режима отображения:**

#### Месяц
- Видны все дни месяца в виде сетки
- Мини-карточки постов внутри каждой даты
- На карточке: время, платформа (TT/IG), тема, цель (цветной бейдж)
- Клик на карточку → модальное окно с полной информацией

#### Неделя
- 7 колонок для текущей недели
- Подробные карточки: время, платформа, цель, тема, формат
- Удобно для планирования текущей недели
- Клик на карточку → модальное окно

#### Список
- Таблица со всеми постами
- Колонки: Дата, Время, Платформа, Цель, Тема, Статус, Действия
- Сортировка по дате
- Кнопка "Открыть" → полный редактор поста

#### Канбан
- Посты сгруппированы по статусам:
  - Идея
  - Промпт готов
  - Текст готов
  - Визуал готов
  - Запланировано
  - Опубликовано
- Удобно для отслеживания прогресса

### 4. Модальное окно поста

**Когда открывается:**
- Клик на мини-карточку в календаре (режим Месяц или Неделя)
- Клик на строку в таблице (режим Список)
- Клик на карточку в канбане

**Что внутри:**
- Основная информация: дата, время, платформа, формат, цель, статус, метрика
- Идея поста
- CTA (призыв к действию)
- Хэштеги (5 штук)

**Кнопки действий:**

1. **Скопировать промпт**
   - Если промпт ещё не создан, генерирует его автоматически
   - Копирует в буфер обмена
   - Показывает "Скопировано" на 2 секунды
   - Используйте для вставки в ChatGPT/Claude/Gemini

2. **Редактировать**
   - Переход на полную страницу редактора
   - Там можно редактировать все поля
   - Добавить аналитику после публикации
   - Вставить ответ от ИИ

3. **Перегенерировать**
   - Создаёт новые SEO-ключи и хэштеги
   - Генерирует новый промпт
   - Меняет статус на "Промпт готов"
   - Полезно если первый вариант не понравился

4. **Изменить дату/время**
   - Открывает форму с выбором даты и времени
   - Сохраняет изменения
   - Пост автоматически перемещается в календаре

5. **Удалить**
   - Запрашивает подтверждение
   - Удаляет пост из всех списков
   - Закрывает модальное окно

### 5. Работа с промптами и ИИ

**Шаг 1:** Откройте пост в модальном окне
**Шаг 2:** Нажмите "Скопировать промпт"
**Шаг 3:** Откройте ChatGPT, Claude или Gemini
**Шаг 4:** Вставьте промпт (Ctrl+V / Cmd+V)
**Шаг 5:** Получите ответ от ИИ
**Шаг 6:** Нажмите "Редактировать" в модальном окне
**Шаг 7:** Вставьте ответ ИИ в поле "Ответ ИИ / Готовый текст"
**Шаг 8:** Измените статус на "Текст готов"
**Шаг 9:** Сохраните

### 6. Статусы постов

Пост проходит через следующие статусы:

1. **Идея** — только базовая информация (по умолчанию при создании)
2. **Промпт готов** — сгенерирован промпт для ИИ (автоматически при генерации плана)
3. **Текст готов** — получен ответ от ИИ, текст готов
4. **Визуал готов** — подготовлены изображения/видео для поста
5. **Запланировано** — пост запланирован в планировщике платформы
6. **Опубликовано** — пост опубликован, можно добавлять аналитику

Меняйте статусы вручную в редакторе поста по мере выполнения работы.

### 7. Добавление аналитики

**Когда:** после публикации поста на платформе

**Как:**
1. Откройте пост через календарь или список
2. Нажмите "Редактировать"
3. Прокрутите до раздела "Аналитика"
4. Нажмите "Добавить результаты"
5. Введите метрики из статистики платформы:
   - Просмотры
   - Лайки
   - Комментарии
   - Сохранения
   - Репосты
   - Подписки
   - Заявки (DM)
   - Переходы в профиль
6. Приложение автоматически рассчитает:
   - Engagement Rate
   - Save Rate
   - Share Rate
   - Comment Rate
   - Subscribe Conversion
   - Lead Conversion
7. Сохраните

### 8. Просмотр аналитики

**Путь:** Боковое меню → "Аналитика"

**Что показывает:**
- Общие метрики: просмотры, лайки, комментарии, сохранения
- Средние показатели: Engagement Rate, Save Rate, Share Rate
- Лучшие темы (по среднему Engagement)
- Лучшие форматы (TikTok Video, Reels, Carousel...)
- Лучшие дни недели
- Лучшее время публикации
- Конверсия: подписки и заявки

Используйте эти данные для улучшения следующих контент-планов.

### 9. Банк идей

**Путь:** Боковое меню → "Банк идей"

**Для чего:**
- Быстро записывать идеи когда они приходят
- Добавлять теги для группировки
- Потом превращать идеи в посты

**Как добавить идею:**
1. Нажмите "+ Добавить идею"
2. Введите название и описание
3. Добавьте теги (через запятую): TikTok, Reels, процесс, ошибки...
4. Сохраните

Позже можно превратить идею в полноценный пост вручную через "Создать пост".

### 10. Экспорт и импорт данных

**Путь:** Боковое меню → "Настройки"

**Экспорт:**
- Нажмите "Экспортировать данные в JSON"
- Файл скачается: `art-content-planner-YYYY-MM-DD-HH-mm.json`
- Храните как резервную копию

**Импорт:**
- Нажмите "Импортировать данные из JSON"
- Выберите ранее экспортированный файл
- Все данные будут восстановлены

**Сброс:**
- Нажмите "Сбросить все данные"
- Подтвердите действие
- Все данные будут удалены (используйте с осторожностью!)

### 11. Цветовая система целей

Цели контента имеют цветовую кодировку для быстрой навигации:

- 🔵 **Охват** (синий) — виральный контент, остановка скролла
- 🟣 **Вовлечение** (фиолетовый) — лайки, комментарии, сохранения
- 🟢 **Доверие** (зелёный) — экспертность, подписки
- 🟠 **Заявка** (оранжевый) — получение DM с вопросами
- 🔴 **Продажа** (красный) — прямая продажа картины или услуги

Эти цвета видны на:
- Мини-карточках в календаре
- Карточках в режиме "Неделя"
- Бейджах в модальном окне
- Бейджах в таблице списка

### 12. Горячие клавиши и советы

**Советы по использованию:**

1. Генерируйте план на месяц в начале месяца
2. Используйте режим "Неделя" для текущей работы
3. Копируйте промпты и создавайте контент заранее
4. Меняйте статусы по мере выполнения
5. Добавляйте аналитику сразу после публикации
6. Экспортируйте данные раз в неделю для безопасности
7. Смотрите аналитику в конце месяца для улучшения следующего плана

**Рекомендуемый график работы:**

- **Начало месяца:** генерация плана на месяц
- **Раз в неделю:** создание контента для постов недели (копирование промптов, работа с ИИ)
- **Каждый день:** публикация постов согласно календарю
- **После публикации:** добавление аналитики в течение 24 часов
- **Конец месяца:** просмотр аналитики, корректировка стратегии
